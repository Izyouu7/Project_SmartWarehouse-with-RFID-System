ติดตามการสแกน RFID จาก Raspberry Pi แบบ Real-time
LIVE — อัพเดททุก 5 วินาที
🥧 Raspberry Pi — วิธีส่งข้อมูลมายัง API
Endpoint สำหรับ Raspberry Pi ส่งข้อมูล RFID scan:
POST http://<เซิร์ฟเวอร์ IP>:3000/api/rfid/scan
Header: x-api-key: rpi_warehouse_rfid_key_2026
Content-Type: application/json

Body JSON:
{
  "tag_code": "RFID-A01-001",
  "status": "In-Stock",    // In-Stock | Moving | Shipped
  "reader_id": "READER-A",
  "signal_strength": -55,
  "location_hint": "Zone-A"  // optional
}
Python ตัวอย่าง (Raspberry Pi):
import requests
requests.post('http://SERVER_IP:3000/api/rfid/scan',
  headers={'x-api-key': 'rpi_warehouse_rfid_key_2026'},
  json={'tag_code': 'RFID-001', 'status': 'In-Stock', 'reader_id': 'RPI-1'})
